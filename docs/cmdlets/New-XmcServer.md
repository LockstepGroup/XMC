# New-XmcServer

## SYNOPSIS
Creates a new XmcServer object.

## SYNTAX

```
New-XmcServer [<CommonParameters>]
```

## DESCRIPTION
Creates a new XmcServer object.

## EXAMPLES

### Example 1
```powershell
PS C:\> New-XmcServer
```

Creates a blank XmcServer object for use with other cmdlets.
